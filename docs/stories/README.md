# Stories

Story specs live here, one file per story. See the `doccraft-story` skill
(installed under `.claude/skills/doccraft-story/SKILL.md` or
`.cursor/skills/doccraft-story/SKILL.md`) for the YAML frontmatter contract,
tag vocabulary, and enum values.

This is a flat folder — no epic subfolders. Grouping happens via prefixed
tags (`area:`, `slice:`, `theme:`) in story frontmatter.

The authoritative story index and prioritisation live in
[`../backlog.md`](../backlog.md) under **Story files (incremental migration)**.
